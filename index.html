개인정보 처리방침

본 개인정보 처리방침은 환율 계산기 앱인 Xchange(개발자: 김동하)에 적용됩니다.

1. 수집하는 개인정보 항목

Xchange 앱은 사용자의 개인정보를 직접 수집하지 않습니다.

2. 개인정보의 수집 방법

해당 앱은 별도의 회원가입, 로그인 또는 사용자 입력 기능이 없으며, 자동 또는 수동으로 어떤 개인정보도 수집하지 않습니다.

3. 개인정보의 이용 목적

개인정보를 수집하지 않으므로 별도의 이용 목적이 존재하지 않습니다.

4. 개인정보의 보관 및 파기

개인정보를 저장하거나 서버에 보관하지 않으며, 수집이 없기 때문에 파기도 필요하지 않습니다.

5. 제3자 제공 및 외부 서비스

당사는 사용자 정보를 제3자에게 제공하지 않으며, 외부 광고나 분석 도구를 사용하지 않습니다. 환율 정보는 외부 환율 API(예: ExchangeRate API 등)를 통해 가져오되, 사용자 개인 정보를 포함하지 않습니다.

6. 사용자의 권리

사용자는 언제든지 앱을 삭제하여 사용을 중단할 수 있으며, 앱은 어떤 개인정보도 수집하거나 저장하지 않기 때문에 별도의 삭제 요청은 필요하지 않습니다.

7. 개인정보 보호조치

앱은 개인정보를 저장하거나 전송하지 않으므로 보안 리스크가 없습니다. 또한 Google Play의 개인정보 보호 가이드를 준수합니다.

Privacy Policy

This Privacy Policy applies to the currency calculator app Xchange, developed by Kim Dongha.

1. Personal Data Collected

The Xchange app does not collect any personal information from users.

2. Method of Data Collection

The app does not include sign-up, login, or any user input functionality, and does not collect any data either manually or automatically.

3. Purpose of Data Use

There is no specific purpose of data use as no personal data is collected.

4. Data Storage and Disposal

No data is stored or maintained on a server, and therefore, there is no need for disposal.

5. Third-Party Sharing and Services

The app does not share user information with any third party, nor does it use external advertising or analytics tools. Currency data is retrieved through external APIs (e.g., ExchangeRate API), and it does not include any personally identifiable information.

6. User Rights

Users may delete the app at any time to stop its use. Since no personal data is collected or stored, there is no need to request data deletion.

7. Data Protection Measures

Since the app does not store or transmit any personal data, there are no security risks. It complies with Google Play's privacy protection guidelines.
